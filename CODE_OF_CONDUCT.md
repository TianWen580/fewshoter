# Code of Conduct

## Our Pledge

We are committed to making participation in this project a harassment-free experience for everyone.

## Our Standards

- Be respectful and constructive
- Welcome different viewpoints and backgrounds
- Focus on what is best for the community
- Show empathy toward other contributors

## Unacceptable Behavior

- Harassment, personal attacks, or discriminatory language
- Publishing private information without permission
- Any conduct that is inappropriate in a professional setting

## Enforcement

Report unacceptable behavior by opening a private security report on GitHub.
Project maintainers will review and respond appropriately.
